# Release Notes

Dieser Ordner enthält die dauerhaft versionierten Texte der öffentlichen
Produkt-Releases. Der wöchentliche Release-Prozess erzeugt für jede Version
genau eine Datei nach dem Muster `v0.21.0.md`.

Dieselben strukturierten Inhalte werden in den Changelog und die
In-App-Versionshistorie übernommen. Ein veröffentlichter Release wird niemals
nachträglich überschrieben; Korrekturen erhalten eine neue Patch-Version.

Die internen PoC-Release-Candidates (`poc-v…-rc.…`) gehören nicht in diesen
Produktkanal und werden weiterhin getrennt behandelt.
