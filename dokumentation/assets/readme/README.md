# README-Bildassets

Diese Bilder sind versionierte Dokumentationsassets und keine Laufzeitquellen der Anwendung.

- `versorgungs-kompass-header-v2.svg`: skalierbarer README-Header mit den vier Produktmodulen.
- `versorgungs-kompass-module-collage-v2.png`: gerenderte Collage aus realen Browserzuständen aller vier Produktmodule.
- `versorgungs-kompass-module-collage-v2.html`: reproduzierbare Browserquelle der Collage.
- `module-previews/`: Browserquellen der vier Collage-Kacheln.
- `markenkit-preview.html`: visuelle Übersicht der Logo-Varianten und Modulfarben.

Die sichtbaren Personen, Organisationen und Fachdaten sind fiktiv. Eine Nutzung als offizielles gematik-Marketingmaterial benötigt eine separate Freigabe.

Neue Versionen erhalten einen neuen, eindeutigen Dateinamen. Frühere README-Bilder aus der Zeit vor der Trennung von öffentlicher Demo und geschützter Anwendung werden nicht aus der Git-Historie restauriert.
